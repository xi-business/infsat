Examples *_fin.inf are examples included in tests and are supposed to be finite.
Examples *_inf.inf are examples included in tests and are supposed to be infinite.
The rest of *.inf examples are not included in tests.
Examples *_fin.py and *_inf.py generate infsat input on stdout and should be run with python3 and
piped into infsat. Usually their sole argument is size of infsat input. The postfix describes if
the example is supposed to be finite or infinite.
